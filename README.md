# Habit Tracker (React)

A simple and clean habit tracker web app built with React. This project helps users build consistency by tracking daily habits, marking progress, and visualizing completion in an intuitive interface. Designed for learning React fundamentals while producing a practical, real-world app.

## Features

- Add, edit, and delete habits
- Mark habits as completed for the day
- Track daily progress per habit
- Clean and minimal UI
- Fully responsive layout
- State management using React hooks

## Built With

- React
- JavaScript (ES6+)
- HTML5
- CSS3

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

- Node.js (v16 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/your-username/habit-tracker-react.git
```

2. Navigate to the project folder
```bash
cd habit-tracker-react
```

3. Install dependencies
```bash
npm install
```

4. Start the development server
```bash
npm run dev
```

5. Open your browser and visit
```
http://localhost:5173
```

3. Edit or delete habits as needed
4. Track your consistency daily
